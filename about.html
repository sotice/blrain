你好，我是 blrain。这是我的随笔——一个用于记录、存档与创作的地方。我喜欢把想法写下来，把读过的书留住，把值得回味的东西存档。

这个网站是一次对个人空间的探索：在喧嚣的互联网里，给自己留一块安静的书桌。没有算法，没有推送，只有文字和它所承载的思考。

如果你对我的代码和项目感兴趣，可以在 GitHub 上找到我：github.com/sotice。

保持好奇，继续阅读，继续写。– blrain

关于这个网站

这是一个静态网站，手工编写，没有后端，没有数据库，没有追踪脚本。页面风格受早期个人主页启发——温润的纸张纹理，衬线字体，柔和的色调。

网站使用 HTML、CSS 和一点 JavaScript 手写而成，字体来自 Google Fonts。所有内容存放在静态文件中，可以部署到任何静态托管服务上。
